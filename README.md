# Brand CRM

A full-stack Brand CRM app (Next.js + Node.js/Express + MongoDB)  
Includes: Login, Dashboard, File Uploads.

---

## Requirements

- Node.js 16+
- npm 9+ (or yarn)
- MongoDB running locally
